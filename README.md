# Pong (with) AI

A version of pong using the love 2d game engine and lua scripting.

The game can be played with 2 players, 1 player or no players.

## Controls

- Player 1 - left hand side paddle :
  - w - up
  - s - down
- Player 2 - right hand side paddle :
  - upArrow - up
  - downArrow - down
